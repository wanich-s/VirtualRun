รายละเอียดกิจกรรมเดิน - วิ่งสะสมระยะทาง “พิบูลบำเพ็ญเกมส์” Virtual Run

ขอเชิญชวนผู้ปกครอง นักเรียน บุคลากร และบุคคลทั่วไป ของแต่ละสถาบันโรงเรียนสาธิตฯ เข้าร่วมชิงชัยในกิจกรรมเดิน-วิ่งสะสมระยะทาง “พิบูลบำเพ็ญเกมส์” Virtual Run ผ่านระบบแข่งขันประเภททีม สังกัดโรงเรียนสาธิตฯใดจะคว้าถ้วยรวมระยะทางสูงสุดไปครอง โดยมีวัตถุประสงค์
1. เพื่อเสริมสร้างความสัมพันธ์ระหว่างภายนอกและภายในโรงเรียนสาธิตฯ แต่ละสถาบัน
2. เพื่อเป็นการหารายได้สนับสนุนการจัดการแข่งขันกีฬาสาธิตสามัคคี ครั้งที่ 45 “พิบูลบำเพ็ญเกมส์”
3. เพื่อส่งเสริมการออกกำลังกายเพื่อรักษาสุขภาพ

  • เปิดรับสมัคร :
  • 27 กันยายน - 20 ตุลาคม พ.ศ. 2565
  • ช่วงเวลาในการเดิน/วิ่งสะสมระยะทาง :
  • เริ่ม 1 - 31 ตุลาคม พ.ศ. 2565 (23.59 น.)
  • ประกาศผล :
  • วันที่ 9 พฤศจิกายน พ.ศ. 2565 และมอบโล่รางวัลในพิธีปิดกีฬาสาธิตสามัคคีฯ
  • (สถาบันที่ได้ระยะทางรวมสูงสุด 3 อันดับแรก ยกเว้นเจ้าภาพ)

รางวัล :

ประเภทบุคคล
เมื่อสมัครเข้าร่วมโครงการฯ ได้รับเสื้อที่ระลึก 1 ตัว
เดิน/วิ่งสะสมระยะทางครบ 45 กิโลเมตร รับเหรียญรางวัล
ประเภททีม
โล่รางวัลสำหรับสถาบันที่ได้ระยะทางเดิน/วิ่ง รวมสะสมสูงสุด 3 อันดับแรก
(ตามสังกัดโรงเรียนสาธิตฯ)
  • ติดตามข่าวสาร ถาม-ตอบคำถาม ผ่าน Facebook Fanpage :
  • StvirtualRun45

กติกา :

  1. เมื่อสมัครร่วมกิจกรรม(ค่าสมัคร 450 บาท) ได้รับเสื้อเป็นที่ระลึก โดยสมัคร เดิน/วิ่ง สะสมระยะทางและส่งผลการเดิน/วิ่งผ่านลิงค์ที่กำหนด ผู้เข้าร่วมแข่งขันสามารถส่งผล เดิน/วิ่ง ย้อนหลังได้ไม่เกิน 3 วัน
  2. การส่งผล เดิน/วิ่ง ตามหมายเลข e-BIB ของแต่ละท่านสามารถวิ่งหรือเดิน ในทุกที่ ทุกเวลา ในสนาม ทางเดินธรรมชาติหรือบนลู่วิ่งจับระยะทางด้วยนาฬิกา หรือแอพมือถือ แล้วแคปเจอร์หน้าจอ หรือถ่ายภาพแสดงผลลู่วิ่งไฟฟ้า ที่แสดงผลการวิ่งที่ระบุวัน ช่วงเวลา และระยะทางการเดินหรือวิ่งอย่างครบถ้วน เพื่อส่งระยะการวิ่ง/เดิน ส่งผลออนไลน์เป็นวิดีโอและภาพถ่าย สำหรับการสะสมระยะทางท่านสามารถใช้แอพพลิเคชั่นที่เก็บสะสมระยะทาง ที่แสดงระยะทาง ตำแหน่ง วัน เวลา สถานที่ และระบุตัวตนของท่าน เช่น Endomondo, Strava, Nike Run Club, Adidas Running App, Runkeeper, Google Fit, Pedometer, Pacer, Run with Map My Run, PUMATRAC เป็นต้น
  3. ผลการเดิน/วิ่ง ต้องเป็นผลที่เกิดขึ้นหลังจากวันที่เริ่มแข่งขันแล้วเท่านั้น ผลการวิ่ง/เดิน ไม่สามารถตกแต่ง ดัดแปลง (หากตรวจพบการทุจริตทางฝ่ายจัดขอตัดสิทธิ์การเข้าร่วมการแข่งขัน)
  4. ผลการตัดสินประเภทบุคคล คำนวณจาก ระยะทางเดิน/วิ่ง สะสมของแต่ละบุคคล เมื่อครบระยะทางสะสม 45 กิโลเมตร ได้รับเหรียญรางวัลเป็นที่ระลึก
  5. ผลการตัดสินประเภททีม(ตามสังกัดโรงเรียนสาธิตฯที่สมัคร) คำนวณจาก ระยะทางเดิน/วิ่ง สะสมสูงสุด 3 อันดับแรก ยกเว้นเจ้าภาพ (คำนวณจากผลรวมระยะทางสะสมของแต่ละบุคคลในทีมนั้นๆ)
  6. จัดส่งของรางวัลในช่วงเดือน พฤศจิกายน – ธันวาคม พ.ศ. 2565
ส่งหลักฐานการชำระเงิน
  1. โอนเงินไปบัญชีที่ระบุด้านล่างเต็มจำนวน โดยไม่หักค่าธรรมเนียมการโอน
  2. เก็บหลักฐานการโอนเงิน พร้อมทั้งถ่ายภาพไว้
  3. แจ้งข้อมูลการโอนเงินผ่านเว็บไซต์ โดยแจ้ง วัน เวลา และจำนวนเงิน พร้อมทั้งแนบรูปภาพหลักฐานการการโอน
  4. เจ้าหน้าที่จะใช้เวลาตรวจสอบ 1-2 วันทำการ และจะยืนยันการชำระเงินในระบบให้คุณทราบ
